Project Overview

The declining population of bees due to global warming.
What would happen if bees disappeared?
Without bees, crops would fail to bear fruit.
If that happens, humans would no longer be able to consume
fruits and vegetables. To help solve this issue, we launched
an upcycling project.

Project Background

In the face of the worsening issue of global warming,
we initiated an “upcycling project.”
Through this program, we created upcycled furniture, promoted it through a web page,
and included English, the global language, to spread awareness about protecting the Earth worldwide.

Upcycling Definition

Upcycling is the process of creatively recycling discarded or seemingly useless items and resources
to give them new value and purpose. Unlike simply recycling materials back into raw components,
upcycling aims to create higher-quality or newly valuable products.
